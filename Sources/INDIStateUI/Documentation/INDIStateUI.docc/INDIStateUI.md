# ``INDIStateUI``

INDIStateUI provides SwiftUI wrappers for INDI state management.

## Overview

`INDIStateUI` provides SwiftUI integration for INDIKit:

- **@Observable Wrappers**: Observable types for SwiftUI views
- **SwiftUI Views**: Pre-built views for displaying INDI devices and properties
- **Reactive Updates**: Automatic UI updates when device state changes
