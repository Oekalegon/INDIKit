# ``INDIState``

INDIState provides state management for INDI devices and properties.

## Overview

`INDIState` builds on top of `INDIProtocol` to provide state management capabilities:

- **Device State**: Track connected devices and their properties
- **Property State**: Maintain current values and states of properties
- **Connection Management**: Manage connections to INDI servers
- **State Synchronization**: Keep local state in sync with server state
